# Prompt 93: Thread Spinner from Long Content

**Category:** Content Creation from the Web
**Where to Use:** Any long-form article
**ROI Level:** Medium | **Setup:** Easy

## What This Does
Converts any long article into a ready-to-post social media thread for X or LinkedIn. Pulls out the strongest points, writes a scroll-stopping hook, structures numbered value points, and closes with an engagement driver - all character-optimised for the platform.

---

## Copy & Paste Prompt

```
# Turn This Article Into a Social Media Thread

Read this article and create a thread for [X / LinkedIn].

## My Details
- Niche: [YOUR NICHE]
- Brand: [YOUR BRAND]
- Target audience: [YOUR AUDIENCE]

## Thread Structure
- Post 1: Hook - bold claim, surprising stat, or contrarian take (max 280 chars for X)
- Posts 2-8: One key insight per post, numbered
- Post 9: Summary or "here's what this means for you"
- Post 10: Engagement close - question, CTA, or repost prompt

## Rules
- Extract only the most valuable, actionable points
- Each post must stand alone but flow as a sequence
- Write in first person - this is my take, not a summary
- No hashtags in the thread body (add 3-5 at the end for LinkedIn only)
- Keep each post under 280 characters for X / under 500 characters for LinkedIn
- Open loops between posts to keep people reading
- Add a line break between the hook line and the supporting text in Post 1

## Format
Label each post clearly: [1/10], [2/10], etc.
```

---

## What You'll Get Back
- A complete 10-post thread ready to copy and paste
- A hook post designed to stop the scroll and earn clicks
- Numbered value posts that each deliver a standalone insight
- An engagement close that drives comments and shares
- Platform-optimised character counts for X or LinkedIn

## Pro Tips
- Post the hook as a standalone post first to test engagement before dropping the full thread
- Screenshot the thread and repurpose it as a carousel for Instagram
- Run this on your own long-form content to squeeze more value from every piece you publish
