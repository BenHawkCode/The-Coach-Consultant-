# Prompt 21: Meta Ad Library Scraper

**Category:** Ad Intelligence
**Where to Use:** Meta Ad Library search results page (facebook.com/ads/library)
**ROI Level:** High | **Setup:** Easy

## What This Does
Extracts and organises every visible ad from a Meta Ad Library search into a structured competitive intelligence report. You get a complete breakdown of ad formats, copy patterns, offers, and creative approaches - saving hours of manual ad research.

---

## Copy & Paste Prompt

```
# Meta Ad Library Intelligence Scrape

Analyse every ad visible on this Meta Ad Library search results page. For each ad, extract and organise the following:

## Per-Ad Breakdown
- Advertiser name
- Ad format (single image, video, carousel, collection)
- Primary text (first 3 lines of copy)
- Headline and description
- Call-to-action button text
- Landing page URL if visible
- Start date (how long it's been running)
- Whether it appears to have multiple variations

## Pattern Analysis
After cataloguing each ad, identify:
1. The most common ad formats being used
2. Recurring copy structures (problem-agitate-solve, direct offer, story-based, etc.)
3. Most frequently used CTAs
4. Common offer types (free lead magnet, discount, webinar, direct sale)
5. Creative approaches (lifestyle imagery, text-heavy, UGC style, professional production)
6. Emotional triggers being targeted (fear, aspiration, urgency, social proof)

## Competitive Summary
Rank the top 3 strongest ads based on: clarity of offer, strength of hook, and how long they've been running. Explain why each one is strong.

Format everything in a clean table first, then the analysis sections below.

My niche: [YOUR NICHE]
My offer: [YOUR OFFER]
```

---

## What You'll Get Back
- Complete table of every ad with format, copy, CTA, and run dates
- Pattern analysis showing what formats and copy structures dominate the niche
- Offer type breakdown revealing what competitors are selling and how
- Creative approach summary identifying the visual styles that are getting used
- Top 3 strongest ads with reasoning you can reverse-engineer
- Emotional trigger mapping across the competitive landscape

## Pro Tips
- Search for your exact niche keywords first, then search for specific competitor names separately - you'll get different intelligence from each
- Sort by "longest running" in the Ad Library before running this prompt - long-running ads are almost always profitable ones
- Run this monthly on the same search terms to track how competitor messaging shifts over time
