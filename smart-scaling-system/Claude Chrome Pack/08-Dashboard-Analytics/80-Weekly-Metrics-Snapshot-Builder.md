# Prompt 80: Weekly Metrics Snapshot Builder

**Category:** Dashboard & Analytics Reading
**Where to Use:** Any analytics dashboard
**ROI Level:** High | **Setup:** Easy

## What This Does
Builds a structured weekly business metrics snapshot from whatever analytics dashboard you're viewing. Pulls out the key numbers, identifies trends, highlights wins and concerns, and gives you 3 priority actions for the coming week. Works with any dashboard - GA, social, CRM, Stripe, or anything else.

---

## Copy & Paste Prompt

```
# Weekly Business Metrics Snapshot

You are reading my analytics dashboard. Build a structured weekly metrics snapshot I can review in under 2 minutes.

## Build the following report:

### Key Numbers This Week
- List every important metric visible on this dashboard
- Show the actual number and the direction (up/down/flat vs. last period)
- Flag any metric that changed by more than 15% in either direction
- Format as a simple table: Metric | This Week | Change | Status

### Trends
- What's improving over the last few weeks?
- What's declining?
- What's been flat and might need a shake-up?
- Any patterns emerging that I should pay attention to?

### Wins
- Top 3 things that went well this week based on the data
- What caused each win?
- How can I replicate or amplify these?

### Concerns
- Top 3 things that need attention based on the data
- How serious is each one? (minor, moderate, urgent)
- What happens if I ignore each one for another week?

### 3 Priority Actions for Next Week
- Based on everything above, give me exactly 3 actions
- Rank them by impact
- Make each one specific enough that I can do it without further planning
- Format: Action | Why | Expected Impact

My business is [YOUR BRAND] and my niche is [YOUR NICHE].
Keep the entire snapshot under 400 words - this is a quick weekly review, not a deep dive.
```

---

## What You'll Get Back
- Clean metrics table with direction indicators and change flags
- Trend summary showing what's improving, declining, or stagnant
- Top 3 wins with replication strategies
- Top 3 concerns with severity ratings and consequences of inaction
- Exactly 3 prioritised actions for next week with expected impact
- The entire report in under 2 minutes of reading time

## Pro Tips
- Run this every Monday morning on your most important dashboard to start the week focused
- Save each snapshot in a running document so you can spot month-over-month patterns
- Use this on multiple dashboards in sequence to build a complete weekly business review in under 10 minutes
