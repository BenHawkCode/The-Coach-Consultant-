# Claude Chrome Resource Pack

## What This Is
A sellable resource pack of 100 copy-paste prompts for the Claude Chrome extension. Each prompt is designed to be pasted directly into Claude Chrome while browsing a specific type of webpage.

## Pack Structure
- **00-Onboarding/** - Setup guides, brand definition, master index, progress tracker
- **01-10 Category Folders** - 10 prompts per category, 100 total
- Each prompt file contains: description, copy-paste prompt, expected output, pro tips

## Standards
- British English throughout (organisation, behaviour, optimise)
- All prompts are generic enough for any business owner or service provider
- Placeholders use [SQUARE BRACKETS] and CAPS: [YOUR BRAND], [YOUR NICHE], [YOUR AUDIENCE], [YOUR OFFER]
- No AI-sounding phrases in any copy
- Prompts are specific and actionable, not vague
- Every prompt tells Claude Chrome exactly what to analyse on the current page

## Core Placeholders (defined once in 03-DEFINE-YOUR-BRAND-FIRST.md)
- [YOUR BRAND] - Business or personal brand name
- [YOUR NICHE] - Industry or market (e.g. "fitness coaching for busy professionals")
- [YOUR AUDIENCE] - Target customer description
- [YOUR OFFER] - Core product or service
- [YOUR WEBSITE] - Business website URL
