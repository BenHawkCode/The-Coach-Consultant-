# How Claude Chrome Works

## What Claude Chrome Can See

When you open Claude Chrome on any webpage, it can read:

- All visible text on the page
- Headlines, paragraphs, lists, buttons, links
- Navigation menus and footer content
- Form fields and their labels
- Image alt text (descriptions attached to images)
- Table data and structured content
- Meta information (page titles, descriptions)

## What Claude Chrome Cannot See

- Content behind login walls (unless you're logged in and it's visible on screen)
- Images themselves (it reads alt text, not the actual image)
- Content that loads only when you scroll down (load it first by scrolling)
- Other browser tabs or windows
- Your local files or computer
- Pop-ups that haven't loaded yet

## How to Get the Best Results

### 1. Load the Full Page First
If the page has "load more" buttons, infinite scroll, or expandable sections - open them before asking Claude to analyse. It can only read what's currently loaded in the page.

### 2. Be Specific About What You Want
"Analyse this page" works. But "Analyse this page and give me a pricing comparison table with their packages, features, and price points" works better. The prompts in this pack are already written to be specific.

### 3. Stack Your Prompts
You can ask follow-up questions in the same conversation. After getting a competitor analysis, ask: "Now write me 3 Instagram hooks based on the gaps you identified." Claude remembers the context from your previous messages in that session.

### 4. Use It on Different Page Types
The same competitor might have a homepage, about page, services page, blog, and testimonials page. Run different prompts on each one, then ask Claude to combine the intelligence into a single report.

### 5. Replace the Placeholders
Every prompt in this pack uses placeholders like [YOUR NICHE] and [YOUR BRAND]. Replace these with your actual details before pasting. Or better yet, fill in **03-DEFINE-YOUR-BRAND-FIRST.md** once and keep it handy.

## What This Is NOT

This is not Claude Projects (a persistent workspace with memory and uploaded files).
This is not Claude Cowork (an autonomous agent that builds deliverables).
This is not Claude Code (a developer tool for writing software).

This is Claude Chrome - a browser companion that reads whatever page you're on and responds to your questions about it. Think of it as having a strategist sitting next to you while you browse the internet, analysing everything you look at in real time.
