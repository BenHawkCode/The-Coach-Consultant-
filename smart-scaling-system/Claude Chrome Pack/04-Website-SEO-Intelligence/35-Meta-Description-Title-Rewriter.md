# Prompt 35: Meta Description & Title Rewriter

**Category:** Website & SEO Intelligence
**Where to Use:** Any of your website pages
**ROI Level:** Medium | **Setup:** Easy

## What This Does
Analyses your current meta titles and descriptions, identifies why they are underperforming, and rewrites them for higher click-through rates. Balances SEO keyword targeting with compelling copy that makes searchers choose your result over competitors on the same results page.

---

## Copy & Paste Prompt

```
# Meta Description & Title Rewriter

Analyse the meta title and meta description of this page. Then rewrite them to maximise click-through rate from Google search results while maintaining SEO relevance.

## Current Page Analysis
- Extract the current meta title and description from this page
- Identify the primary topic and likely target keyword
- Assess: Is the current title compelling enough to click? Does the description sell the click?

## Rewrite Rules
- Meta titles: 50-60 characters max. Front-load the primary keyword. Include a benefit or hook
- Meta descriptions: 150-160 characters max. Specific outcome or benefit. End with a reason to click now
- Do NOT use clickbait or misleading phrasing
- Include power words that drive action without sounding desperate
- Every version must read naturally - not keyword-stuffed

## My Details
- My brand: [YOUR BRAND]
- My target audience: [YOUR AUDIENCE]
- Primary keyword for this page: [TARGET KEYWORD IF KNOWN, OR "DETERMINE FROM PAGE CONTENT"]

## Output Format
1. Show the current meta title and description with a brief critique of each
2. Write 3 meta title alternatives - each with a different angle (benefit-led, curiosity, direct)
3. Write 3 meta description alternatives to pair with the best title option
4. Recommend your top title + description combination and explain why it wins
5. Bonus: Suggest 2-3 secondary keywords to weave into the page content to support the meta tags
```

---

## What You'll Get Back
- A critique of your current meta title and description with specific weaknesses identified
- Three meta title alternatives using different persuasion angles
- Three meta description alternatives optimised for click-through rate
- A recommended winning combination with reasoning
- Secondary keyword suggestions to strengthen on-page SEO
- Character counts confirmed for every option so nothing gets truncated

## Pro Tips
- Run this on every page of your site - meta tags are the fastest SEO win with the least effort
- Check Google Search Console for pages with high impressions but low CTR - those are your priority pages
- Update meta tags and wait 2-4 weeks before measuring the impact in Search Console
