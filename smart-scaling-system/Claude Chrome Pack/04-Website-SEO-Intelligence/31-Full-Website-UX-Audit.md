# Prompt 31: Full Website UX Audit

**Category:** Website & SEO Intelligence
**Where to Use:** Any website homepage
**ROI Level:** High | **Setup:** Easy

## What This Does
Runs a comprehensive UX audit on any website - covering navigation, layout, mobile-friendliness, accessibility, user flow, and friction points. You get specific, prioritised fixes rather than vague suggestions, so you know exactly what to change and why it matters for conversions.

---

## Copy & Paste Prompt

```
# Full Website UX Audit

Analyse this entire webpage as a UX specialist. I need a brutally honest audit covering every element that affects user experience and conversion.

## Audit Areas
- Navigation: Is it intuitive? Can users find what they need in 2 clicks or fewer?
- Layout & visual hierarchy: Does the eye flow naturally to the most important elements?
- Load signals: Are there heavy images, excessive scripts, or render-blocking elements visible in the page structure?
- Mobile-friendliness: Based on the layout and elements, would this work well on a phone screen?
- Accessibility: Contrast ratios, alt text presence, heading structure, button sizing
- User flow: Map the journey from landing to conversion - where does it break down?
- Friction points: Identify every element that could cause hesitation, confusion, or abandonment

## My Details
- My niche: [YOUR NICHE]
- My target audience: [YOUR AUDIENCE]
- The primary action I want visitors to take: [YOUR OFFER / PRIMARY CTA]

## Output Format
1. Score each audit area out of 10 with a one-line justification
2. List the top 5 critical fixes (highest impact on conversions)
3. List 5 secondary improvements (nice-to-haves that compound over time)
4. For each fix, give me the EXACT change to make - not just "improve the CTA" but specifically what to change and where
```

---

## What You'll Get Back
- A scored breakdown of every UX category with specific justifications
- The top 5 highest-impact fixes ranked by conversion potential
- Secondary improvements that compound results over time
- Exact, implementation-ready changes for each recommendation
- Friction point identification with suggested alternatives
- A clear priority order so you know what to tackle first

## Pro Tips
- Run this on your own site first, then run it on your top competitor's site to benchmark
- Re-run after making changes to track improvement across each category
- Pay closest attention to the user flow analysis - that is where most conversions are lost
