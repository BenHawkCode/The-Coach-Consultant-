# Roles & Responsibilities

## [Creative Director]
**Owns:** Content type selection, hooks, topics, captions, CTAs, YouTube titles, thumbnail text, resource CTAs, YT script docs
**Workflow:** Fills in creative columns in Instructure, writes/approves scripts in weekly script files

## [Content Manager]
**Owns:** Date scheduling, day assignment, time slots, editing QA, final check, posting/scheduling to platforms
**Workflow:** Schedules content calendar, QAs edited content, distributes to Drive delivery folders

## [Automation Lead]
**Owns:** ManyChat CTA setup, automation flows, DM triggers
**Workflow:** Sets up ManyChat keywords when CTA changes, tests automation flows before content goes live

---

## Handover SOP

### Weekly Content Cycle
1. [Creative Director] completes creative columns in Instructure files at least 7 days ahead
2. [Creative Director] writes full scripts in the relevant weekly script file (2026/[Month]/[Week]/)
3. [Content Manager] assigns dates and types, sends to editing team
4. Editing team creates visuals/edits based on scripts and editor notes
5. [Content Manager] QAs: checks hook matches script, caption complete, CTA correct
6. [Content Manager] marks Edited > Checked > Posted/Scheduled
7. [Automation Lead] ensures ManyChat keywords are live before posting
8. [Content Manager] uploads final content to the correct Google Drive delivery folder

### Content Approval Flow
Script written > Edited > Checked > Posted

No content goes live without all three checkboxes ticked.
