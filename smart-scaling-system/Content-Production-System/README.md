# Content Production Operating System

## What This Is
Complete content planning, scripting, and distribution system. Covers 4 channels: Personal Media AM Posts, Personal Media PM Posts, Company Social Media, and YouTube. Built to be scheduled, planned, and handed over to an editor and distributor.

## System Structure

### instructure/
Master content management sheets - one per channel. This is where content is planned, tracked, and QA'd.
- Personal-Media-AM-Post-Instructure.md
- Personal-Media-PM-Post-Instructure.md
- Company-Social-Media-Instructure.md
- YouTube-Instructure.md
- Creative-SOP.md (how to use the system)

### Channel Folders (Delivery)
Where final content files are managed before distribution.
- Personal-Media-AM-Posts/
- Personal-Media-PM-Posts/
- Company-Social-Media-Posts/
- YouTube-Files/

### Asset-Bank/
Central storage for all production assets.
- Image-Bank/ - Personal brand photos
- Video-Bank/ - BTS footage and b-roll
- Client-Result-Bank/ - Testimonials, transformations, results
- Login-Distribution-Inspiration.md - Platform logins, distribution links, creative references

### 2026/
Full year of weekly script, caption, and hook templates organised month-by-month, week-by-week.

### Roles
- **[Creative Director]** - Content type, hooks, topics, captions, CTAs, scripts
- **[Content Manager]** - Scheduling, QA, editing, distribution
- **[Automation Lead]** - ManyChat, automation flows
