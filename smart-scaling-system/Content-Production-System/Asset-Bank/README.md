# Asset Bank

## Overview
Central storage for all content production assets. Keep this organised so the editing team can find what they need without asking.

## Folders

### Image-Bank/
All personal brand photos for use in content. Talking shots, lifestyle, studio, events, etc.
- **Naming:** [context]-[number].jpg (e.g. studio-001.jpg, speaking-012.jpg, lifestyle-003.jpg)
- **Rule:** Never reuse the same image twice in the same month
- **Tip:** After each photoshoot, batch rename and drop all usable images here

### Video-Bank/
Raw BTS footage, b-roll clips, talking head clips, and any reusable video content.
- **Naming:** [date]-[description].mp4 (e.g. 2026-04-07-bts-office.mp4)
- **Organised by:** Shoot date
- **Tip:** Keep raw footage separate from edited clips. Label edited clips with "-edited" suffix

### Client-Result-Bank/
Screenshots, testimonials, before/afters, DM conversations, case studies.
- **Naming:** [client-name]-[type]-[date] (e.g. sarah-testimonial-2026-03.png, james-transformation-2026-01.jpg)
- **Types:** screenshot, testimonial, transformation, dm-conversation, case-study
- **Rule:** Get permission before using any client content. Keep a permissions log

### Login-Distribution-Inspiration.md
Template for all platform logins, distribution channel links, and creative inspiration references. Fill this in during onboarding.
