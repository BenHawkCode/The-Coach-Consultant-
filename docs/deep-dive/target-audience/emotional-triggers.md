## How to Deploy Triggers

### Layering Strategy
- **Hook:** FOMO + Curiosity (grab attention)
- **Body:** Shame + Pride + Social Proof (build tension + aspiration)
- **Close:** Urgency + Hope + Trust (drive action)

### Content Type Mapping
- **Social Media:** FOMO + Curiosity + Quick Social Proof
- **Email Campaigns:** Shame + Hope + Trust (longer narrative)
- **Sales Pages:** All 10 triggers layered systematically
- **YouTube:** Curiosity + Belonging + Achievement journey

### Frequency Guidelines
- Primary trigger per piece: 1 (clear focus)
- Supporting triggers: 2-3 maximum
- Trust/credibility: Always present (foundation layer)
