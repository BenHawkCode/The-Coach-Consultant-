## Usage Guidelines

**When generating content:**
1. Read quick reference files first (voice-characteristics, alex-morgan-persona, core-beliefs)
2. Reference FULL files when deep context needed for authentic voice matching
3. Always maintain British English and voice characteristics
4. Target Alex Morgan persona pain points
5. Use emotional triggers strategically

**File Naming Convention:**
- `-FULL.md` = Full sample content (200-500 lines) extracted from Google Drive
- `.md` = Quick reference summary created from IP folder analysis
